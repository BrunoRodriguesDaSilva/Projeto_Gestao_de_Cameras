# 📹 GestaCam - Gestão de Ativos de CFTV

O **GestaCam** é uma aplicação completa para o inventário e monitoramento de câmeras de segurança. Ele permite centralizar informações técnicas (como endereços IP) e acompanhar a saúde do sistema em tempo real através de um painel interativo.

---

## 🚀 Funcionalidades Principal
*   **Dashboard Inteligente:** Gráfico de rosca dinâmico que mostra a porcentagem de câmeras Ativas, Inativas ou em Manutenção.
*   **CRUD Completo:** Interface para Adicionar, Listar, Editar e Excluir câmeras com persistência de dados.
*   **Busca em Tempo Real:** Filtro instantâneo por nome ou IP e filtragem por categorias de status.
*   **Autenticação Segura:** Tela de login protegida para acesso administrativo.
*   **Design Moderno:** Interface responsiva e otimizada para Dark Mode.

---

## 🛠️ Tecnologias Utilizadas

### Frontend:
*   **Tailwind CSS:** Estilização moderna e rápida via classes utilitárias.
*   **JavaScript (ES6+):** Lógica de consumo de API e manipulação do DOM.
*   **Chart.js:** Visualização de dados dinâmica.

### Backend:
*   **Node.js & Express:** Servidor robusto e gerenciamento de rotas.
*   **SQLite3:** Banco de dados relacional leve e sem necessidade de configuração externa.
